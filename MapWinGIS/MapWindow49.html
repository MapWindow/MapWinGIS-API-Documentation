This is demo of the MapWinGIS ActiveX control. Similar custom GIS application can be built based on this app or on the barebone MapWinGIS control in a matter of days. Here is the source code of this project.

The development of this project depends on the public donations. Our plans include further development of MapWinGIS core library and also releasing a new version of MapWindow desktop GIS based on it, which will be called MapWindow 5. This demo can be considered the first version of it. Below is an overview of perspective features for each project.

Basic architecture:

a) .NET wrappers for MapWinGIS classes;
  • more object oriented (Layer, Feature classes);
  • the use of .NET interfaces (IEnumerable, IDisposable);
  • possible support of GeoAPI library;
b) Flexible plug-in and GIS toolbox interfaces using MEF;
c) Written completely in C#/.NET 4.5;
d) ReSharper and StyleCop compliant;

Existing functionality to be reworked under new architecture:
  • Table Editor (Syncfusion's Grid control);
  • Projections support, including easy search by EPSG-code;
  • GIS toolbox based on MEF interfaces;
  • Printing / layout plug-in;
  • Legend control;
  • Symbology editor;
  • Tiles support;
New functionality:
  • Make the ocx registry-free using a manifest file as mentioned here;
  • Make a ClickOnce deployment with automated updates;
  • GUI to set up visualization of raster layers;
  • Data repository/browser for files, databases and servers;
  • New add layer GUI, to set option before loading;
  • Add logging using Log4Net or NLog;
 Improvement of the existing functionality:
  • Further integration with OGR (better support of large layers);
  • Asynchronous rendering for large datasources;
  • Extending Unicode support (it's not complete yet);
  • Add support for the new GeoPackage format, which is already supported bij GDAL/OGR;
  • Implement missing GDAL/OGR tools, like gdal_grid.exe
Support of various GIS Web Services:
  • Web Feature Service (WFS);
  • Web Map Service (WMS);
  • Web Coverage Service (WCF);
  • Web Map Tile Service (WMTS);
  • Catalog Service for the Web (CSW);
